  - 营养信息可视化

- 多媒体资源跳转

## 版本说明

### 版本号格式
版本号格式为 `主版本号.次版本号.修订号`，递增规则如下：

- **主版本号**：当你做了不兼容的 API 修改
- **次版本号**：当你做了向下兼容的功能性新增
- **修订号**：当你做了向下兼容的问题修正

### 更改类型

- **新增** - 新功能
- **更改** - 对现有功能的更改
- **弃用** - 即将移除的功能
- **移除** - 已移除的功能
- **修复** - 问题修复
- **安全** - 安全相关的修复

### 日期格式
所有日期使用 ISO 8601 格式 (YYYY-MM-DD)
