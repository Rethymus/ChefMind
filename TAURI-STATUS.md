### 🖥️ GUI显示问题
由于在WSL2环境中运行，原生GUI窗口无法显示。这是预期的行为，因为：
- WSL2默认没有GUI环境
- 需要额外的X11服务器配置
- 或者在原生Linux环境中运行

### 🚀 可用的解决方案

#### 方案1: Web版本（推荐用于WSL2）
```bash
npm run dev
```
访问: http://localhost:1420

#### 方案2: 原生Linux环境
在有GUI的Linux系统中运行：
```bash
npm run tauri:dev
```

#### 方案3: WSL2 + X11转发
安装X11服务器并配置DISPLAY变量
